# The Quiet Struggle

A youth-led classroom project on teen mental health by Kaylee.

## Pages

- **Home** (`index.html`)
- **Editor's Note** (`editors-note.html`)
- **Flash Memoir** — *My Struggle with Mental Health* (`flash-memoir.html`)
- **Information Effect** — *What Do We Know?* (`information-effect.html`)
- **Persuasion Effect** — *How Can We Help?* (`persuasion-effect.html`)
- **Resources** (`resources.html`)
- **Sources** (`sources.html`)

## Local preview

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000/>.

## Tech

Static HTML + CSS, hand-drawn SVG illustrations, Google Fonts (Fraunces + Karla), no build step.

## Help

If you or someone you know is struggling, call or text **988** anytime — free, confidential, 24/7. Visit [988lifeline.org](https://988lifeline.org).
